# Convolution and Connective Component of Sparse Matrices

## Usage
```
>>> # Basic
>>> A = spMatrix(2) # the dimension is 2
>>> A
{}
>>> A[1,2],A[-10,2]
(0, 0)
>>> A[1,2],A[-10,2] = 1,1
>>> A
{'[-10, 2]': 1, '[1, 2]': 1}
>>>
